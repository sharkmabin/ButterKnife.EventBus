# ButterKnife.EventBus

ButterKnife 和 EventBus 第三方库的简单使用演示项目。

## 功能

- ButterKnife 视图绑定：通过 @BindView 注解绑定控件，@OnClick 处理点击事件
- EventBus 事件通信：演示 String 字符串、自定义对象 MsgBean、Integer 整数三种消息类型的发送与接收
- EventBus 注册与反注册：在 onCreate 注册，onStop 反注册

## 技术栈

- Java / Android
- ButterKnife 8.5.1
- EventBus

## 说明

这是一个学习/演示项目，适合 Android 初学者了解 ButterKnife 视图注入和 EventBus 事件总线的基本用法。
